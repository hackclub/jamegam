# jame gam

Landing page for [jamegam.hackclub.com](https://jamegam.hackclub.com).

> a community of teenagers who enter game jams together, run by hack club. build a game, get prizes. free, international, no experience needed.

Built with SvelteKit, deployed on Vercel.

## Develop

```bash
bun install
cp .env.example .env   # fill in the secrets
bun run dev
```

```bash
bun run build          # production build
```
